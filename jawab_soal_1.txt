Axios adalah sebuah tools atau library yang digunakan dalam pengembangan web app untuk mengirim request kepada server dan meg-handle respons yang diterima. Dengan menggunakan Axios, pengembang dapat dengan mudah mengirimkan data dari aplikasi ke server dan menerima data dari server kembali ke aplikasi. Hal ini sangat berguna dalam pengembangan aplikasi web yang memerlukan komunikasi dengan server untuk mengambil atau mengirimkan data.
